# haskell-lsp

Haskell Language Server (HLS) for Claude Code. Provides type checking,
diagnostics, and code intelligence for Haskell projects.

## Supported Extensions
`.hs`, `.lhs`

## Installation

### Via GHCup (recommended)
```bash
ghcup install hls
```

### Requirements
- GHC version matching your project (managed by GHCup or Stack)
- A `hie.yaml`, `cabal.project`, or `stack.yaml` in the project root
